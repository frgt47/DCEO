# 📈 StockInsight Dashboard (주식 현황 파악 대시보드)

> **"주가만 보는 투자는 이제 그만! 데이터로 시장의 흐름을 읽으세요."**

StockInsight Dashboard는 주식 초보자부터 중급 투자자까지, 여러 사이트를 돌아다닐 필요 없이 **시장의 핵심 지표를 한눈에 파악**할 수 있도록 설계된 전문 투자 보조 도구입니다.

---

## 🌟 주요 특징 (Key Features)

### 1. 🔍 지능형 키워드 스캐너
복잡한 종목 검색 대신, `joosik.txt`에 정의된 핵심 키워드를 조합하여 나만의 우량주를 발굴합니다.
*   **가치 평가(PER):** 기업이 벌어들이는 이익 대비 주가가 적정한지 필터링합니다.
*   **심리 지표(RSI):** 현재 시장이 너무 뜨거운지(과매수), 혹은 과하게 식었는지(과매도)를 분석합니다.
*   **수급 분석:** 시장의 '큰손'인 외국인의 매수 흐름을 추적합니다.

### 2. 📊 심층 분석 인터랙티브 차트
단순한 선 그래프가 아닌, 투자 결정에 필수적인 기술적 분석을 제공합니다.
*   **이동평균선(MA):** 주가의 평균적인 흐름을 보여주어 상승/하락 추세를 파악하게 해줍니다.
*   **실시간 변동 시뮬레이션:** 시장 지수(KOSPI, KOSDAQ)의 움직임을 실시간으로 시각화합니다.

### 3. 📰 NLP 뉴스 키워드 분석
인공지능(NLP) 기술을 활용하여 방대한 뉴스 속에서 현재 종목의 **핵심 키워드와 긍정/부정 신호**를 추출해 보여줍니다.

## 💡 초보자를 위한 핵심 용어 사전

주식 공부가 처음이신가요? 이 프로그램이 보여주는 지표들의 의미는 다음과 같습니다.

| 용어 | 의미 | 투자 포인트 |
| :--- | :--- | :--- |
| **PER (주가수익비율)** | 이익에 비해 주가가 얼마나 비싼가? | 낮을수록 저평가된 우량주인 경우가 많습니다. |
| **RSI (상대강도지수)** | 주식의 매수/매도 세기 | 70 이상이면 너무 과열(매도 고민), 30 이하면 과매도(매수 고민). |
| **이동평균선 (MA)** | 일정 기간 동안의 주가 평균치 | 주가가 이평선 위에 있으면 보통 '상승 추세'로 봅니다. |
| **ROE (자기자본이익률)** | 내 돈으로 얼마나 효율적으로 돈을 벌었나? | 높을수록 기업이 장사를 잘하고 있다는 뜻입니다. |

## 🚀 시작하기 (How to Use)

1.  이 리파지토리를 클론(Clone)하거나 다운로드합니다.
2.  `stock.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
3.  좌측 **[키워드 스캐너]**에서 원하는 조건을 설정하고 **[종목 스캔 시작]** 버튼을 누르세요.
4.  발굴된 종목의 상세 차트와 펀더멘털 지표를 확인하며 전략을 세웁니다.
